## Description
Alice GlowShop is a beginner-friendly e-commerce web application for makeup and beauty products. Customers can browse products, add them to their shopping cart, increase or decrease quantities, and remove items as needed. The application simulates a real-world online makeup store.

---

## Features
- Browse products with images, name, and price.
- Add products to the cart.
- Increase or decrease quantity of items in the cart.
- Remove items from the cart.
- Dynamic cart total calculation.
- Responsive design using Bootstrap.
- Navigation between pages using React Router.

---

## Technology Stack
- **Frontend:** React, JavaScript, Bootstrap, React Router
- **Backend:** Node.js + Express 
- **Database:** PostgreSQL 
- **Package Manager:** npm
